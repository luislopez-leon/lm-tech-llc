# L&M Tech LLC — Portfolio & Landing Page

> Static landing page for L&M Tech LLC, hosted via GitHub Pages.

## Live Site

🌐 [https://luislopez-leon.github.io/lm-tech-llc](https://luislopez-leon.github.io/lm-tech-llc)

## Stack

- Plain HTML + CSS (no build step, no dependencies)
- Dark-themed, mobile-responsive
- Ready for GitHub Pages deployment

## Local Preview

Just open `index.html` in your browser — no server needed.

## Deploying Updates

```bash
git add .
git commit -m "Update landing page"
git push origin main
```

GitHub Pages auto-deploys on every push to `main`.
